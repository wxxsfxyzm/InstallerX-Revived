# 应用设置

应用设置为您提供了丰富的自定义选项，让您可以根据自己的使用习惯和偏好来调整应用。

## 主题

自定义应用的外观和感觉。

### UI 引擎
InstallerX Revived 提供了两套UI引擎供您选择：
*   **Material Design 3 (MD3)**: Google 最新的设计语言，提供现代化、动态的界面体验。默认使用 Material 3 Expressive 风格，如果您不喜欢，可以在主题设置中回退到标准的 MD3 风格。
*   **MIUIX**: 小米 HyperOS 的设计风格，提供不同的视觉和交互感受。请注意，[MIUIX 库](https://github.com/compose-miuix-ui/miuix)正处于积极的开发和优化中，UI 可能会随时改动，遇到相关问题可以随时向我反馈。

### 动态取色
应用支持动态取色功能，该功能适用于所有安卓版本。它会从您的壁纸中提取颜色，并将其应用于整个应用界面，创造出与系统和谐统一的视觉效果。

您还可以进一步自定义颜色行为：
*   **多彩对话框**: 启用后，安装对话框的颜色将从被安装应用的图标中提取，使得每个应用的安装界面都独具特色。
*   **多彩实时通知进度**: 在 Android 16 及以上版本，启用此选项可以让实时通知中的安装进度条颜色也跟随应用图标的颜色。请注意，您必须先启用应用的“实时通知”功能才能设置此选项。

### 图标偏好
您可以选择应用列表中的图标来源：
*   **解析软件包**: 直接从 APK 文件中解析并显示原始图标。
*   **使用系统图标包**: 如果您安装了第三方图标包，可以选择此项，让应用列表中的图标与您的系统图标保持一致。本功能在 HyperOS 和 OneUI 上测试通过。

### 隐藏应用图标
您可以选择隐藏 InstallerX 的桌面图标。请注意，由于系统限制，在 Android 10 以上的系统版本，应用不再被允许隐藏桌面图标，所以在部分系统上可能隐藏图标后，桌面或应用程序抽屉仍会显示一个指向系统设置应用详情页的图标，你可以在 LSPosed 的设置中禁用此项限制。

## 安装选项

配置全局的安装行为和参数。

### 全局授权器
此处设置的授权器（如 Root 或 Shizuku）将被应用用于执行需要特权的全局操作，例如锁定默认安装器、修改 ADB 验证设置等。用户配置文件也可以沿用此设置来执行安装。

### 全局安装方式
在这里，您可以选择一个全局的默认安装方式（如 Root、Shizuku），此设置可被配置文件沿用。同时，您也可以对安装过程中的 UI 行为进行自定义。

#### 自定义设置解释
*   **自定义安装确认对话框**: 启用后，将使用应用内置的对话框替代系统默认的安装确认框。
*   **安装完成时显示 Toast**: 安装成功或失败后，会在屏幕底部显示一个简短的提示消息。
*   **安装后打开应用**: 安装成功后，提供一个快捷按钮以立即打开刚刚安装的应用。

### 安装来源预设
您可以预设一些常用的安装来源（应用包名）。在为应用创建配置文件或在安装对话框中时，可以方便地快捷选择，简化操作流程。

### 黑名单
通过黑名单可以阻止特定应用的安装。您可以根据应用的**包名**或**共享 UID** 来添加规则。任何匹配黑名单规则的应用都将被拦截，无法安装。

## 卸载器设置

配置与应用卸载相关的选项。

### 授权器说明
卸载器始终使用您在“全局默认配置”中设置的**授权器**来执行卸载操作，以确保拥有必要的权限。

### 卸载选项
*   **保留应用数据**: 卸载应用但保留其数据和缓存文件。
*   **降级安装 (保留数据)**: 将应用降级到旧版本，同时保留用户数据。
*   **冻结/解冻应用**: 临时禁用或重新启用应用，而不是完全卸载。

**注意**: “降级安装”和“冻结/解冻应用”是互斥的，因为它们是两种不同的操作，不能同时执行。

### 非 Root 快捷方式
对于非 Root 用户，由于系统限制无法像 Root 用户一样锁定系统卸载器，InstallerX 提供了一个便捷的快捷方式：您只需输入目标应用的包名，即可快速调用系统的卸载流程。
